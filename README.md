# Slate JS Editor

Slate is a completely customizable framework for building rich text editors.

## Tech Stack

**Client:** React,NextJS and Chakra UI

## Installation

Install Project with yarn

```bash
  git clone https://github.com/naufaldi/slate-js-editor.git
  cd slate-js-editor
  yarn Install
```
